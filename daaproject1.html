0/1 Knapsack

In the 0/1 Knapsack problem, you can either take an entire item or leave it ‚Äî no partial items allowed. It's solved using Dynamic Programming to maximize the total value without exceeding the weight limit.

Fractional Knapsack

In the Fractional Knapsack problem, items can be broken into smaller parts. It's solved using a Greedy Algorithm, usually by selecting items with the highest value-to-weight ratio first.

  • ‚úÖ Items can be partially included.
  • ü•§ Ideal for resources like liquids or grains that can be divided.

üìñ How to Use

  1. Go to the main backpack manager page.
  2. Use the form to add item name, weight, and value.
  3. Click "Add Item" to insert the item into the backpack list.
  4. Choose the knapsack type (0/1 or Fractional) and enter capacity.
  5. Click "Solve" to view the optimal selection of items.
  6. Export the current backpack list using "Export as JSON/CSV" buttons.
  7. Use the clear/reset button to start fresh if needed.
